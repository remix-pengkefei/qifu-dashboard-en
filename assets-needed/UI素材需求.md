参考视觉资产：`/Users/fly/Desktop/奇富展厅/06_4号主控网站视觉资产/`

整体风格：深蓝主调 + 银灰结构 + 青蓝链路 + 极少量绿色 + 极少量红色提示。
国家级金融科技作战指挥中心。
不要赛博朋克 / 不要紫色霓虹 / 不要营销网站感。

### 资产 1 · 中国主地图（卫星夜光版）

**文件名**：`china-map-glow.png`
**尺寸**：3200 × 2200 px（4K 适配，留 12% 透明边距）
**格式**：PNG 透明背景
**位置**：总览态中区主背景

**画面要求**：
- 中国大陆 + 台湾 + 海南，行政边界线发青蓝色光（边线宽 1.5–2px，外发光 8px）
- 内部填充：深蓝海底色（#0a2348）+ 城市群亮点（按真实城市群密度分布：长三角、珠三角、京津冀、成渝、武汉、西安、青岛、福州、长株潭、中原、关中等）
- 城市亮点是金黄/暖白小光斑（直径 3–6px），形成"夜间灯火"密度图，亮度从沿海/经济发达带向西部递减
- 省界青蓝色（#56c4ff）半透明描边，类似 NASA 夜光卫星图风格
- 有轻微的弧形高光感（像看一颗星球的局部）
- **不要**任何文字、地名标注（地名我会在前端用代码标）
- **不要**周边国家 / 海面背景，只要中国轮廓发光，其余全透明
- 整体气质：参考奇富资产 `06_4号主控网站视觉资产/03_局部模块图/04_总览态主战情区模块图.png` 的中国地图主视觉

**用途**：作为中区背景图，其上叠加节点、链路、文字（前端 SVG/CSS 处理）
