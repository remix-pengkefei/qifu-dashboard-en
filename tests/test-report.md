## 注意事项

1. **headless 预览的定时器节流**：长时间空闲后 Chrome 会节流隐藏标签页的 setTimeout，导致看似时钟"停了"。在用户实际打开浏览器使用时不会出现。reload 后立即恢复正常。

2. **中国地图节点位置**：`public/geo/china.json` 来自阿里 DataV.GeoAtlas，36 个一级行政区。城市节点位置是经纬度硬编码（src/data/regions.ts），均为常用坐标。

3. **银行 Logo**：当前用「色块 + 中文字符 + 行名」的占位，整体风格匹配视觉方案（即 SPDB / CEB 等英文行号 + 浦/光等汉字）。如需换成真实彩色 Logo，把 `Bank.color` + `Bank.glyph` 字段换成 SVG 图片即可。

4. **模式切换不重置时钟和指标**：只 reseed 事件流和 counters，让"运行连续感"不被打断。

5. **风险事件概率**：约 7% 触发 risk 级别，11% 触发 warn，24% 触发 ok，余下 normal。
